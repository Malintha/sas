About Ms.Use

This is a IOT based Mobile system used to reduce violence done against women in Sri Lanka. User will be getting a wearable devise (ring) with a bluetooth transmitter and a switch attached to the devise. When the user is facing violance then she can click the swithch in the device immidiatly and mobile application in the phone will be activated and send messages to the nearest people who are using this application. Not only that a message will be sent to the nearest police station. Also this mobile app contains a AR segment. Users can use this to identify the victim without alarming the abuser. Users just have to lift their phones and scan the area like they are recording or taking a picture. Users will be able to see a tag on top of the victim and it will be easy for the users to catch the culprit.
