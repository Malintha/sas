to Stop Violence on Women

This is a IOT, AR and Android based plateform where women can use to be protected from any type of abuse and harrasement

Wearable

In the solution we are providing, user will get a ring with a bluetooth module and a switch. When the user is facing a insidant the user can send a signal to the mobile phone using this ring

Mobile App

This application will allove the user to access the wearable ring. Not only that this app will get all the stress messages sent by users and display them according to the location of the user

AR App

When a person sends the distress message, people who recieves that message don't have to make a fuss over finding the victim and the culprite. They just have to lift the phone with our AR app on. Then the exact location of the victim will be pin pointed

Voice Recorder

When the user is facing an incident of bribary then the user can send a signal to the mobile and a voice recorder will be triggered.This will be hidded from the normal view of the OS of the mobile phone
